# agentbrowser

Chrome for your agent.

<img src="resources/image.jpg">

# Installation

```bash
pip install agentbrowser
```

# Usage

## Importing into your project

```python
from agentbrowser import (
    create_page,
    close_page,
    navigate_to,
    get_body_text,
    get_current_page_id,
    run_until_complete,
    get_current_page,
    switch_to,
    get_document_text,
    get_document_html,
    get_body_html,
    evaluate_javascript,
)
```
